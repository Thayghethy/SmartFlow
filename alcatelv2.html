Como puxar alarme via putty de Alcatel Vivo 2:

Para buscar alarme de cliente vivo 2 através do putty, é necessário primeiramente desassociar o gpon da porta, e com o gpon desassociado prosseguir para acessar o putty. Ao abrir o putty, tem por padrão configurado o ip 200.204.1.4, que é o ip utilizado para o checklist e alarme de vivo 1, o ip para vivo 2 é 10.200.1.29. Ao confirmar o Ip que está configurado, selecionamos "Open"

Em seguida, será aberto a tela do putty, solicitando a matricula, ao preencher a matricula utilizando a letra "a" e o 0 junto do número do RE, exemplo: "a0163046", apertar enter preencher a senha configurada e apertar enter.

Nesta etapa o putty já esta pronto para buscar o alarme.

Para buscarmos o alarme de vivo 2, é necessário o Ip do Armário, que conseguimos no SDU na aba "Informações Rede Interna" ou no Wise, na aba "Informações TBS"

Com o ip do armário, colocamos no Putty o comando "ssh (Ip do armário)" e pressionar enter.
Exemplo:

Digitar "yes" e clicar em enter Colocar a senha de rede e clicar novamente no enter

Após é necessário inserir o seguinte comando "show pon unprovision-onu" e pressionar enter

Após clicar em Enter, será aberto uma planilha, que contém todos os gpons que estão alarmando dentro do Rin que está sendo procurado. é possível identifica o GPON procurando na coluna "SERNUM" e localizar o GPON. Para identificar o alarme de um GPON especifico, primeiro deve-se localiza-lo entre os demais que apareceram. Ao localizar o GPON, é possível identificar o Slot e Porta em que se encontra, olhando na coluna "GPON-INDEX", em que os dois últimos são respectivamente Slot/Porta.

Vale lembrar que todo alarme Alcatel vivo 2, precisa ser somado +4 no slot para identificar a posição.
Para deslogar do armário só digitar "logout" e pressionar enter.

Após localizar o SLOT e a PORTA, você deve converter os mesmos para SPLITTER (ou SP), pois é essa informação que você deve passar para o técnico. LEMBRE-SE QUE NÃO SE DEVE PASSAR SLOT E PORTA PARA O TÉCNICO EM HIPÓTESE ALGUMA. Para saber como realizar a conversão, clique AQUI para o tutorial de conversão.
