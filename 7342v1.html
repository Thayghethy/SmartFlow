Como puxar alarme via putty de Alcatel 7342:

Ao abrir o putty, tem por padrão configurado o ip 200.204.1.4, que é o ip utilizado para o checklist e alarme de vivo 1. Ao confirmar o Ip que está configurado, selecionamos "Open"

Em seguida, será aberto a dela do putty, solicitando a matricula. Ao preencher a matricula, utilizando a letra “a” e o 0 junto do número do RE, exemplo: "a0163050"

Em sequência apertar enter, aceitar o token do duo (no celular), digitar a sua senha de rede e apertar enter

Concluído o login do putty, vai aparecer uma linha com a matrícula no início, semelhante ao exemplo [a0163050@s-br-sp-spo-ctp-jmp-srv-01 ~]$;

Para realizar a busca de alarme de uma OLT 7342 é digitado a palavra "telnet", um espaço colando o nome da OLT, realizar mais um espaço e inserir o numeral 1023. Como no exemplo abaixo:

Pouco após dar Enter no comando é mudado a tela para uma tela preta somente com o símbolo "<", nesse momento deverá apertar Enter e aguardar alguns segundos.

Quando chegar nesta tela inserir a letra "T" clicar no Enter para ser direcionado para a tela de login

Para realizar o login é necessário preencher o Username, que será sempre "operacao" para todos os operadores. Após clicar no enter é solicitado a senha de acesso que sempre é "oper@ultr@1" e apertar Enter.

Após o login realizado, digitar RTRV-ALM-PON::ALL:::,,,,,, e apertar enter para enviar o comando, inserir "Y" e apertar novamente no Enter para confirmar o comando.

Nessa tela se localiza o GPON ao lado de "SERNUM", o slot e a porta do lado de "PON"(os dois últimos números são respectivamente slot/Porta) e o SLID logo ao lado.

Para desconectar da OLT, é usado o comando "logoff", que apresenta em sequência uma confirmação, em que é enviado "Y" para deslogar e "N" para continuar logado.

Após o comando ser reconhecido, o putty já se encontra deslogado da olt e pronto para pesquisar outras caso necessário.

Após localizar o SLOT e a PORTA, você deve converter os mesmos para CABO E PRIMÁRIA, pois é essa informação que você deve passar para o técnico.
